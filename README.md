# Data-Ingestion-Assignment from multiple data sources using python
#### Project Overview
This project demonstrates practical data ingestion from multiple data sources using Python.  
The main objective is to extract, transform, and load (ETL) data from different formats and databases into Python for analysis.

The data sources used in this project include:
- MySQL Database (RDBMS)
- CSV File Format
- XML File Format
- Microsoft Excel File Format
- Apache Parquet File Format
- MongoDB Database (NoSQL)
  ---

## Objectives of the Project
1. To demonstrate data extraction from different data sources.
2. To perform basic data transformation using Python (Pandas).
3. To load and analyze data in Python.
4. To understand differences between structured and unstructured data.
5. To implement ETL processes in real-world scenarios.
